CodeGPT
CodeGPT is a Programming assistant built on Open Ai. The Coding Bot uses text-davinci-003 LLM to generate Code in 15 different languages. The model produces good results out of the box, you must define a language, which you want CodeGPT to respond in. You can then prompt the bot with either; the problem you are trying to solve (in natural language), or the code you have written so far. CodeGPT will depending on your Prompt, produce the lines of code to solve your problem, or Debug the code which you have prompted it with, the code is returned in an IDE friendly format thanks to the help of Prism.js
